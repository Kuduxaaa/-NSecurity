Firewall рЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃЌ рЃЋрЃћрЃЉ РђЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃфрЃЋрЃљрЃерЃў, рЃЋрЃћрЃЉ РђЊ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃАрЃљ рЃЊрЃљ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃА рЃерЃЮрЃарЃўрЃА HTTP рЃбрЃарЃљрЃцрЃўрЃЎрЃўрЃА рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃўрЃЌ рЃЊрЃљ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃЌ.рЃћрЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃгрЃћрЃАрЃў, рЃўрЃфрЃљрЃЋрЃА рЃЋрЃћрЃЉ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃА рЃўрЃАрЃћрЃЌрЃў рЃерЃћрЃбрЃћрЃЋрЃћрЃЉрЃўрЃАрЃњрЃљрЃю, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ

### NSecurity -рЃўрЃА рЃърЃўрЃарЃЋрЃћрЃџ рЃЋрЃћрЃарЃАрЃўрЃљрЃерЃў:

- рЃўрЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ
рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃљрЃа рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ User-Agent. рЃљрЃАрЃћрЃЋрЃћ рЃўрЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃўрЃљрЃю рЃЉрЃЮрЃбрЃћрЃЉрЃў!

- рЃўрЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃљ рЃўрЃА User-Agent рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф
рЃБрЃарЃћрЃЋрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў: ```hydra, sqlmap, w3af, voideye, whatweb
wpscan, nmap, xget, wget, nikto, cisco-torch, 
arachni, cmoix, havij, sqlninja, uil2pn```

- рЃўрЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃљ рЃДрЃЋрЃћрЃџрЃљ SQLinjection, XSS,
RFI, LFI рЃЏрЃфрЃЊрЃћрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃўрЃџрЃЮрЃњрЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃерЃў ```nerve_logs``` рЃљрЃАрЃћрЃЋрЃћ рЃўрЃњрЃќрЃљрЃЋрЃюрЃћрЃЉрЃљ [Telegram](https://telegram.org/) -рЃўрЃА рЃфрЃЕрЃљрЃбрЃерЃў рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ ```рЃерЃћрЃбрЃћрЃЋрЃўрЃА рЃбрЃўрЃърЃў, IP рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў, рЃерЃћрЃбрЃћрЃЋрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў, рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў (User-Agent) рЃЊрЃљ рЃерЃћрЃбрЃћрЃЋрЃўрЃА рЃгрЃћрЃарЃбрЃўрЃџрЃў```

- рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃўрЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃљ
рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃЉрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю TOR -рЃА

### Todo

- рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃърЃљрЃюрЃћрЃџрЃў ­Ъўі рЃћрЃЦрЃюрЃћрЃЉрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃў рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃў рЃАрЃљрЃўрЃЊрЃљрЃюрЃљрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃюрЃА рЃњрЃћрЃЏрЃЮрЃќрЃћ рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮ рЃЊрЃљ рЃЊрЃљрЃљрЃДрЃћрЃюрЃЮ рЃЊрЃљрЃфрЃЋрЃўрЃА рЃЊрЃЮрЃюрЃћрЃћрЃЉрЃў рЃљрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃЊрЃљрЃЉрЃџрЃЮрЃЎрЃЮрЃА sqli рЃерЃћрЃбрЃћрЃЋрЃћрЃЉрЃў рЃЏрЃљрЃњрЃарЃљрЃЏ рЃбрЃЮрЃарЃўрЃА рЃбрЃарЃљрЃцрЃўрЃЏрЃў рЃўрЃДрЃЮрЃА рЃюрЃћрЃЉрЃљрЃЊрЃљрЃарЃЌрЃБрЃџрЃў

- рЃЌрЃБ рЃЊрЃљрЃљрЃЎрЃЋрЃўрЃарЃЊрЃў рЃбрЃћрЃџрЃћрЃњрЃарЃљрЃЏрЃќрЃћ рЃерЃћрЃбрЃћрЃЋрЃўрЃА рЃбрЃўрЃърЃерЃў рЃгрЃћрЃарЃўрЃљ SQLi/XSS/LFI/RFI рЃЋрЃљрЃърЃўрЃарЃћрЃЉ рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃбрЃћрЃЋрЃљрЃА рЃфрЃљрЃџрЃфрЃљрЃЎрЃћ рЃљрЃЊрЃњрЃўрЃџрЃў рЃЏрЃўрЃЋрЃљрЃюрЃўрЃГрЃЮ рЃЊрЃљ рЃЕрЃљрЃЏрЃЮрЃБрЃгрЃћрЃарЃЮ рЃарЃЮрЃЏрЃћрЃџрЃў рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃљрЃЏрЃЮрЃўрЃфрЃюрЃЮрЃА рЃљрЃАрЃћ рЃБрЃцрЃарЃЮ рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃўрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃбрЃћрЃЋрЃўрЃА рЃќрЃБрЃАрЃб рЃЎрЃџрЃљрЃАрЃўрЃцрЃўрЃЎрЃљрЃфрЃЎрЃљрЃА рЃЏрЃЮрЃљрЃ«рЃЊрЃћрЃюрЃА

- рЃБрЃюрЃЊрЃљ рЃерЃћрЃБрЃЦрЃЏрЃюрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ рЃАрЃљрЃЊрЃљрЃф рЃерЃћрЃљрЃњрЃарЃЮрЃЋрЃћрЃЉрЃА рЃАрЃљрЃўрЃбрЃўрЃА рЃЋрЃўрЃќрЃўрЃбрЃЮрЃарЃћрЃЉрЃА рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљ рЃљрЃўрЃърЃў рЃљрЃњрЃћрЃюрЃбрЃў рЃЊрЃљ рЃЏрЃћрЃарЃћ рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃерЃћрЃљрЃ»рЃљрЃЏрЃћрЃЉрЃА рЃЊрЃљ рЃЏрЃЮрЃњрЃфрЃћрЃЏрЃА рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљрЃА

- рЃарЃљрЃЌрЃЦрЃЏрЃљрЃБрЃюрЃЊрЃљ рЃБрЃюрЃЊрЃљ рЃЕрЃљрЃЋрЃљрЃерЃћрЃюрЃЮ рЃарЃЮ рЃбрЃћрЃџрЃћрЃњрЃарЃљрЃЏрЃќрЃћ рЃДрЃЮрЃЋрЃћрЃџрЃў рЃерЃћрЃбрЃћрЃЋрЃўрЃА рЃЏрЃфрЃЊрЃћрЃџрЃЮрЃЉрЃљ рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа рЃЏрЃЮрЃЊрЃўрЃЮрЃЊрЃћрЃА рЃЊрЃљ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЋрЃљрЃЎрЃћрЃЌрЃЮ рЃўрЃАрЃћ рЃарЃЮрЃЏ рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃўрЃЊрЃљрЃю рЃЏрЃЮрЃ«рЃЊрЃћрЃА рЃўрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃЌрЃБ рЃАрЃљрЃЊ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃўрЃњрЃќрЃљрЃЋрЃюрЃЮрЃА рЃџрЃЮрЃњрЃћрЃЉрЃў рЃљрЃюрЃБ рЃарЃЮрЃЏрЃћрЃџ рЃЕрЃљрЃбрЃерЃў ­Ъўі

- рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮ рЃюрЃћрЃарЃЋрЃўрЃА рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃерЃўрЃцрЃарЃљрЃфрЃўрЃљ рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў рЃарЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃљрЃЊрЃљрЃф рЃЉрЃљрЃќрЃљрЃф рЃарЃЮрЃЏ рЃњрЃљрЃбрЃДрЃЊрЃћрЃА рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃњрЃљрЃерЃўрЃцрЃЋрЃарЃљ рЃцрЃљрЃЦрЃбрЃўрЃБрЃарЃљрЃЊ рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ

- рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮ рЃљрЃюрЃбрЃў рЃЊрЃЮрЃАрЃў рЃЊрЃљ рЃљрЃ«рЃљрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃЎрЃљрЃюрЃўрЃарЃћрЃЉрЃљ, рЃерЃћрЃџрЃўрЃА рЃФрЃћрЃЉрЃюрЃљ, рЃЋрЃўрЃарЃБрЃАрЃћрЃЉрЃўрЃА рЃАрЃЎрЃљрЃюрЃўрЃарЃћрЃЉрЃљ, рЃърЃЮрЃарЃбрЃћрЃЉрЃўрЃА рЃАрЃЎрЃљрЃюрЃўрЃарЃћрЃЉрЃљ, рЃАрЃљрЃўрЃбрЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў рЃЊрЃљ рЃљ.рЃе

- рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮ IP рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃўрЃА рЃЌрЃћрЃЌрЃарЃў рЃЊрЃљ рЃерЃљрЃЋрЃў рЃАрЃўрЃљ

- рЃЋрЃњрЃћрЃњрЃЏрЃљрЃЋ рЃЕрЃљрЃЋрЃљрЃерЃћрЃюрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃў рЃЊрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃўрЃЊрЃљрЃю рЃерЃћрЃњрЃћрЃФрЃџрЃЮрЃА рЃЉрЃљрЃќрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃгрЃљрЃерЃџрЃљ рЃЕрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ рЃљрЃю рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ

- рЃБрЃЎрЃЋрЃћ рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃљрЃЦрЃЋрЃА рЃарЃЮрЃБрЃбрЃћрЃарЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃЋрЃгрЃћрЃарЃЮ рЃАрЃ«рЃЋрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЏрЃќрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃФрЃарЃљрЃЋрЃљрЃЊ рЃЋрЃљрЃЦрЃфрЃўрЃЮ

- рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃерЃўрЃЊрЃљ рЃЦрЃАрЃћрЃџрЃўрЃЊрЃљрЃю рЃљрЃю рЃљрЃЊрЃЏрЃўрЃюрЃўрЃА рЃљрЃўрЃърЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃўрЃЊрЃўрЃю рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃў рЃюрЃћрЃарЃЋрЃўрЃА рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃерЃў рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ

### рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ
рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў рЃБрЃюрЃЊрЃљ рЃерЃћрЃЏрЃЮрЃўрЃбрЃљрЃюрЃЮрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЎрЃЮрЃЊрЃерЃў
Firewall рЃарЃЮрЃњрЃЮрЃарЃф рЃћрЃА рЃ«рЃЊрЃћрЃЉрЃљ ```/autoload.php``` рЃцрЃљрЃўрЃџрЃерЃў
рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃљрЃЦрЃбрЃЮрЃБрЃарЃЮрЃЌ рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
```php
<?php
require_once '/controller/router.php';
$nrv = new Nerve(); // рЃљрЃЦ рЃЌрЃЦрЃЋрЃћрЃю рЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЉрЃЌ Waf -рЃА
```

рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃЋрЃўрЃФрЃџрЃўрЃљрЃЌ (рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃўрЃљ) рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў
рЃБрЃЎрЃЋрЃћ рЃЕрЃљрЃерЃћрЃюрЃЉрЃБрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃБрЃџрЃў рЃарЃЮрЃБрЃбрЃћрЃарЃў, рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃўрЃф
рЃЌрЃЦрЃЋрЃћрЃю рЃБрЃюрЃЊрЃљ рЃерЃћрЃЏрЃЮрЃўрЃбрЃљрЃюрЃЮрЃЌ рЃЎрЃЮрЃЊрЃерЃў рЃарЃЮрЃБрЃбрЃћрЃарЃў
рЃарЃЮрЃБрЃбрЃћрЃарЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
```php
<?php
require_once '/controller/router.php';
// рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃЊрЃћрЃЎрЃџрЃљрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў
Route::add('/', function(){
  echo 'рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃњрЃЋрЃћрЃарЃЊрЃў';
});

// GET рЃарЃЮрЃБрЃбрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў
Route::add('/contact', function(){
  echo '</h1>рЃАрЃљрЃЎрЃЮрЃюрЃбрЃљрЃЦрЃбрЃЮ рЃцрЃЮрЃарЃЏрЃљ</h1><form method="post"><input type="text" name="test" /><input type="submit" value="send" /></form>';
}, 'get');

// POST рЃарЃЮрЃБрЃбрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў
Route::add('/contact', function(){
  print_r($_POST);
}, 'post');

// рЃарЃЮрЃБрЃбрЃў рЃарЃЮрЃЏрЃџрЃўрЃЌрЃљрЃф рЃЋрЃўрЃдрЃћрЃЉрЃЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА
// рЃарЃћрЃњрЃћрЃЦрЃАрЃўрЃА рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃўрЃЌ
Route::add('/news/([0-9]*)',function($id){
  echo 'рЃАрЃўрЃљрЃ«рЃџрЃўрЃА ID: ' . $id;
});
```
